Fork the repository and run the following:
```
npm i
npm start
```  
### Directory structure:

- File Structure:
  - Whole Pages `.jsx` -> `./src/components/screens`
  - Individual Components `.jsx` -> `./src/components`
  - CSS -> Along with the component/page `.jsx` file
  - Images -> `./public/img`
<br>

CSS:
  - Root Variables / Default Values -> `App.css`
